# Drive Music Player

A music player for Google Drive.

## Files supported

| Extension | mimetype      |
|-----------|---------------|
| flac      | audio/flac    |
| m3u       | audio/mpegurl |
| m3u8      | audio/mpegurl |
| m4a       | audio/mp4     |
| m4b       | audio/mp4     |
| mp3       | audio/mpeg    |
| ogg       | audio/ogg     |
| opus      | audio/ogg     |
| pls       | audio/x-scpls |
| wav       | audio/wav     |
| aac       | audio/aa      |

## Authentication Method

Simple email/user and password authentication. Don't use OAuth.
